Detailed Description

La classe Profil contient une chaine de caractères qui est username, une chaine de caractères qui est adresse mail, une chaine de caractères qui est password, avec le nom, le prenom, l'age, le sexe, la ville, le portable , l'instagram, le snapchat, le facebook et le tiktok (les reseaux sont facultatifs), une chaine de caractères qui est type de profil, un tableau de chaines de caractères qui est type de sport, une chaine de caractères qui est lien vers une photo, un tableau des liens sur Avis, un tableau de chaines de caractères qui est liste de followers, un tableau de chaines de caractères qui est liste de following.
